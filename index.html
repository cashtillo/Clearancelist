ℹ️ HOW TO USE RAY'S LIST

🔍

Search

Type any part of the item name or SKU to find a listing instantly.

🖨

View & Print

Tap View & Print on any card to open the original tag, then print from your browser.

📄

Adding a New Listing

Save the approval email to the Ray's List folder. Save the PDF and drop it here — the tool auto-scans all listings inside automatically.

🚫

Sold Out

Tap Sold Out when an item sells through. It hides from the main list and auto-deletes after 7 days. Tap Restore to bring it back.

📋

Audit Mode

PIN-protected. Shows all original tags as one continuous scroll — scan barcodes off the screen or hit Print All. Contact David for access.
